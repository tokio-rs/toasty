Toasty API Documentation — nightly

Toasty is incubating. APIs are evolving and not yet stable.

Core Crates

  • toasty — User-facing API: database handle, query engine, and runtime
  • toasty_core — Shared types: schema representations, statement AST, driver interface
  • toasty_macros — Proc-macro entry points (#[derive(Model)], #[derive(Embed)])

SQL

  • toasty_sql — Statement AST to SQL string serialization, used by SQL drivers

Database Drivers

  • toasty_driver_sqlite — SQLite driver
  • toasty_driver_postgresql — PostgreSQL driver
  • toasty_driver_mysql — MySQL driver
  • toasty_driver_dynamodb — DynamoDB driver
  • toasty_driver_integration_suite — Shared integration test suite for driver implementors to verify their driver

Utilities

  • toasty_cli — Command-line interface
