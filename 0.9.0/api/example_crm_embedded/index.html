crm-embedded: a contact card composed from EMBEDDED value types — a typed id, a flattened address, and a tagged-union contact method. None of these gets its own table; their columns are flattened into the contacts table.

Run it cold (cargo run -p example-crm-embedded). In-memory SQLite by default; set TOASTY_CONNECTION_URL for another backend.
