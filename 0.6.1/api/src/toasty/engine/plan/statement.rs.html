1//! Plans a single HIR statement into MIR operations.
2//!
3//! # Overview
4//!
5//! Each HIR statement produces one or more MIR nodes that form an execution
6//! graph. The planner splits work into two concerns:
7//!
8//! - **Data loading**: issuing the database operation (SQL query, GetByKey, etc.)
9//! - **Output processing**: projecting, filtering, or merging the loaded data
10//!   into the shape the caller expects
11//!
12//! The entry point is `PlanStatement::plan()`, which runs these phases in order:
13//!
14//! 1. Extract columns needed by the returning clause
15//! 2. Discover args (references to other statements) in the filter and
16//!    assignments, registering their MIR node outputs as inputs
17//! 3. Rewrite the statement's assignment/value expressions so their `Arg`
18//!    positions reference `load_data.inputs` indices
19//! 4. Plan the data-loading MIR node (SQL or NoSQL path)
20//! 5. Create projection nodes for back-references (so child statements can
21//!    read columns from this statement's results)
22//! 6. Plan child/dependent statements
23//! 7. Build the output node (projection, eval, or nested merge)
24//!
25//! # How `Expr::Arg` flows through the planner
26//!
27//! `Expr::Arg(n)` appears throughout the statement to reference data from
28//! other statements. The meaning of `n` changes as the expression moves
29//! through the pipeline:
30//!
31//! ## HIR level (statement enters the planner)
32//!
33//! `Arg(n)` indexes into `stmt_info.args`, a list of `hir::Arg` entries
34//! created during lowering. Each entry is either:
35//!
36//! - `Arg::Sub { stmt_id, input, .. }` — data from a child statement
37//!   (e.g., a subquery result used in an `IN` list)
38//! - `Arg::Ref { stmt_id, data_load_input, .. }` — a column from a
39//!   parent/sibling statement (back-reference for nested queries)
40//!
41//! At this point, `input` and `data_load_input` are unset `Cell`s.
42//!
43//! ## After `extract_data_load_args`
44//!
45//! This pass walks the filter and assignments, and for each `Arg(n)`:
46//!
47//! - Looks up `stmt_info.args[n]` to find the source MIR node
48//! - Adds that node to `load_data.inputs` (an `IndexSet<NodeId>`)
49//! - Records the index within `load_data.inputs` back into the `hir::Arg`
50//!   cell (`input.set(Some(index))` or `data_load_input.set(Some(index))`)
51//!
52//! The `Arg(n)` positions in the statement expressions are unchanged at
53//! this point — they still reference HIR arg positions.
54//!
55//! ## After `rewrite_stmt_*_arg_dependencies`
56//!
57//! These methods rewrite `Arg` nodes in **assignments and insert values
58//! only** (not the filter). They read the cells populated above:
59//!
60//! - `Arg::Sub { input }` → `Expr::arg(input.get())` — now indexes into
61//!   `load_data.inputs`
62//! - `Arg::Ref { data_load_input, batch_load_index }` →
63//!   `Expr::arg_project(data_load_input, [batch_load_index, column])`
64//!
65//! After this, assignment expressions have MIR-level arg positions. Filter
66//! expressions still have HIR-level positions.
67//!
68//! ## During MIR node construction (`rewrite_expr_for_mir`)
69//!
70//! When building MIR nodes that carry their own expressions (Guard, Project
71//! for key expressions), the expression may reference a subset of the
72//! statement's `load_data.inputs`. `rewrite_expr_for_mir` does two things:
73//!
74//! 1. Resolves each `Arg(hir_pos)` through `stmt_info.args[hir_pos]` to
75//!    find the `load_data.inputs` index and MIR node ID
76//! 2. Assigns a new compact position (0, 1, 2, ...) and rewrites the
77//!    `Arg` node in place
78//!
79//! It returns the arg types and input node IDs for constructing the MIR
80//! node. This is the same resolution as `rewrite_arg_dependencies` but
81//! builds a compact, node-specific input set rather than reusing the full
82//! `load_data.inputs` list.
83//!
84//! ## At execution time
85//!
86//! Each MIR node's `to_exec()` maps its input `NodeId`s to `VarId`s. The
87//! executor loads variables by `VarId` and passes them to `eval::Func`,
88//! which resolves `Arg(n)` against the provided input slice.

148impl HirPlanner<'_> {
149    pub(super) fn plan_statement(&mut self, stmt_id: hir::StmtId) -> Result<()> {
150        let stmt_info = &self.hir[stmt_id];
151
152        // Check if the statement has already been planned
153        if stmt_info.load_data_statement.get().is_some() {
154            return Ok(());
155        }
156
157        // First, plan independent dependency statements. These are statments
158        // that must run before the current one but do not reference the current
159        // statement.
160        for &dep_stmt_id in &stmt_info.deps {
161            if self.hir[dep_stmt_id].independent {
162                self.plan_statement(dep_stmt_id)?;
163            }
164        }
165
166        let stmt = stmt_info.stmt.as_deref().unwrap().clone();
167
168        // Delegate to PlanStatement
169        let mut planner = PlanStatement {
170            planner: self,
171            stmt_id,
172            stmt_info,
173            load_data: LoadData {
174                inputs: IndexSet::new(),
175                select_items: SelectItems::new(),
176                batch_load_args: IndexSet::new(),
177            },
178            remaining_deps: stmt_info.deps.iter().cloned().collect(),
179        };
180        planner.plan(stmt)?;
181
182        Ok(())
183    }
184}

189    fn plan(&mut self, mut stmt: stmt::Statement) -> Result<()> {
190        let mut returning = stmt.take_returning();
191
192        // For single VALUES queries (e.g., batch queries), the VALUES body is
193        // the output expression. Extract it as a returning value so the planner
194        // can wire up sub-statement dependencies.
195        if returning.is_none()
196            && let stmt::Statement::Query(query) = &mut stmt
197            && let stmt::ExprSet::Values(values) = &mut query.body
198        {
199            returning = Some(stmt::Returning::Expr(if query.single {
200                assert_eq!(1, values.rows.len());
201                values.rows.drain(..).next().unwrap()
202            } else {
203                stmt::Expr::list(std::mem::take(&mut values.rows))
204            }));
205        }
206
207        // No queries are single at this point.
208        match &mut stmt {
209            stmt::Statement::Query(stmt) => stmt.single = false,
210            stmt::Statement::Insert(stmt) => stmt.source.single = false,
211            _ => {}
212        }
213
214        // Visit the main statement's returning clause to extract needed columns
215        self.extract_columns_from_returning(&returning);
216
217        // Process any args (sub statements or refs to parent statements) in the query's filter.
218        self.extract_data_load_args(&mut stmt);
219
220        // For each back ref, include the needed columns
221        self.collect_back_ref_columns();
222
223        // If there are any ref args, then the statement might need to be
224        // rewritten to batch load all records for a NestedMerge operation.
225        if !self.load_data.batch_load_args.is_empty() {
226            debug_assert!(stmt.is_query());
227            self.rewrite_stmt_for_batch_load(&mut stmt);
228        } else if let stmt::Statement::Insert(insert) = &mut stmt {
229            self.rewrite_stmt_insert_arg_dependencies(insert);
230        } else if let stmt::Statement::Update(update) = &mut stmt {
231            self.rewrite_stmt_update_arg_dependencies(update);
232        }
233
234        let load_data_node_id = self.plan_data_loading(stmt, &mut returning)?;
235
236        // Track the exec statement operation node.
237        self.stmt_info
238            .load_data_statement
239            .set(Some(load_data_node_id));
240
241        // Now, for each back ref, we need to project the expression to what the
242        // next statement expects.
243        self.process_back_ref_projections(load_data_node_id);
244
245        // Track the selection for later use.
246        // TODO: Do we actually need to track this on the statement?
247        self.stmt_info
248            .load_data_select_items
249            .set(mem::take(&mut self.load_data.select_items))
250            .unwrap();
251
252        // Plan each child
253        self.plan_child_statements()?;
254
255        // Track sub-statements referenced in the returning clause as inputs, so their
256        // results are available when building the return value.
257        let returning_info = ReturningInfo {
258            inputs: self.extract_inputs_from_returning(&mut returning, load_data_node_id),
259            clause: returning,
260        };
261
262        // Plans a NestedMerge if one is needed
263        let output_node_id = self.plan_output_node(load_data_node_id, returning_info);
264
265        self.stmt_info.output.set(Some(output_node_id));
266
267        Ok(())
268    }

272    fn extract_inputs_from_returning(
273        &mut self,
274        returning: &mut Returning,
275        load_data_node_id: mir::NodeId,
276    ) -> IndexSet<mir::NodeId> {
277        let mut inputs = IndexSet::new();
278
279        let is_returning_projection = matches!(returning, Some(stmt::Returning::Project(..)));
280        debug_assert!(
281            is_returning_projection || matches!(returning, None | Some(stmt::Returning::Expr(..)))
282        );
283
284        visit_mut::for_each_expr_mut(returning, |expr| {
285            match expr {
286                stmt::Expr::Arg(expr_arg) => {
287                    match &self.stmt_info.args[expr_arg.position] {
288                        hir::Arg::Ref {
289                            stmt_id: target_id,
290                            returning_input,
291                            batch_load_index,
292                            target_expr_ref,
293                            ..
294                        } => {
295                            let target_stmt_info = &self.planner.hir[target_id];
296                            let back_ref = &target_stmt_info.back_refs[&self.stmt_id];
297
298                            // Find the column
299                            let column = back_ref.exprs.get_index_of(target_expr_ref).unwrap();
300
301                            if returning_input.get().is_none() {
302                                // Find the node providing the data for the ref
303                                let node_id = back_ref.node_id.get().unwrap();
304
305                                let (index, _) = inputs.insert_full(node_id);
306                                returning_input.set(Some(index));
307                            }
308
309                            let index = returning_input.get().unwrap();
310                            let row = batch_load_index.get().unwrap();
311                            let nesting = if self.stmt().is_insert() && is_returning_projection {
312                                1
313                            } else {
314                                0
315                            };
316
317                            *expr = stmt::Expr::project(
318                                stmt::ExprArg {
319                                    position: index,
320                                    nesting,
321                                },
322                                [row, column],
323                            );
324                        }
325                        hir::Arg::Sub {
326                            stmt_id: target_id, ..
327                        } => {
328                            assert!(
329                                !(self.stmt().is_insert() && is_returning_projection),
330                                "TODO"
331                            );
332
333                            let target_stmt_info = &self.planner.hir[target_id];
334                            let target_node_id = target_stmt_info.output.get().expect("bug");
335
336                            let (index, _) = inputs.insert_full(target_node_id);
337
338                            *expr = stmt::Expr::arg(index);
339                        }
340                    }
341                }
342                stmt::Expr::Project(expr_project) if !is_returning_projection => {
343                    // When returning an expression (not projection),
344                    // ExprReference projections need to be handled explicitly.
345                    if let stmt::Expr::Reference(expr_reference) = &*expr_project.base {
346                        let [row] = expr_project.projection.as_slice() else {
347                            todo!("expr_projec{expr_project:#?}")
348                        };
349
350                        let column = self.load_data_expr_reference_position(expr_reference);
351                        let (position, _) = inputs.insert_full(load_data_node_id);
352                        *expr = stmt::Expr::arg_project(position, [*row, column]);
353                    }
354                }
355                stmt::Expr::Reference(expr_reference) if is_returning_projection => {
356                    let column = self.load_data_expr_reference_position(expr_reference);
357                    let (position, _) = inputs.insert_full(load_data_node_id);
358                    *expr = stmt::Expr::arg_project(position, [column]);
359                }
360                stmt::Expr::Func(stmt::ExprFunc::Count(stmt::FuncCount { arg: None, .. }))
361                    if is_returning_projection =>
362                {
363                    let index = self
364                        .stmt_info
365                        .load_data_select_items
366                        .get()
367                        .unwrap()
368                        .get_index_of_count_star();
369                    let (position, _) = inputs.insert_full(load_data_node_id);
370                    *expr = stmt::Expr::arg_project(position, [index]);
371                }
372                _ => {}
373            }
374        });
375
376        inputs
377    }

380        assert!(
381            expr_reference.is_column(),
382            "TODO: expr_reference = {expr_reference:#?}"

384
385        let Some(column) = self

391        else {

396        };

400    fn extract_columns_from_returning(&mut self, returning: &Returning) {
401        stmt::visit::for_each_expr(returning, |expr| match expr {
402            stmt::Expr::Reference(expr_reference) => {
403                assert!(
404                    expr_reference.is_column(),
405                    "TODO: expr_reference = {expr_reference:#?}"

407                self.load_data.select_items.insert((*expr_reference).into());
408            }
409            stmt::Expr::Func(stmt::ExprFunc::Count(stmt::FuncCount { arg: None, .. })) => {
410                self.load_data.select_items.insert(SelectItem::CountStar);
411            }
412            _ => {}
413        })

416    /// Extract arguments needed to perform data loading
417    fn extract_data_load_args(&mut self, stmt: &mut stmt::Statement) {
418        if let Some(filter) = stmt.filter() {
419            stmt::visit::for_each_expr(filter, |expr| {
420                self.extract_data_load_args_from_expr(expr, None);
421            });
422        }
423
424        if let stmt::Statement::Insert(insert) = stmt {
425            let stmt::ExprSet::Values(values) = &insert.source.body else {
426                todo!()
427            };
428
429            for (i, row) in values.rows.iter().enumerate() {
430                stmt::visit::for_each_expr(row, |expr| {
431                    self.extract_data_load_args_from_expr(expr, Some(i));
432                });
433            }
434        }
435
436        if let stmt::Statement::Update(update) = stmt {
437            for (_, assignment) in update.assignments.iter() {
438                stmt::visit::for_each_expr(assignment, |expr| {
439                    self.extract_data_load_args_from_expr(expr, None);
440                });
441            }
442        }
443    }

445    fn extract_data_load_args_from_expr(&mut self, expr: &stmt::Expr, insert_row: Option<usize>) {
446        if let stmt::Expr::Arg(expr_arg) = expr {
447            match &self.stmt_info.args[expr_arg.position] {
448                hir::Arg::Sub {
449                    stmt_id: target_id,
450                    returning,
451                    input,
452                    batch_load_index,
453                    ..
454                } => {
455                    debug_assert!(!returning, "the argument was found in a filter");
456
457                    let target = &self.planner.hir[target_id];
458                    let Some(node_id) = target.output.get() else {
459                        panic!(
460                            "bug: expected target statement to be planned; curr={:#?}; target={:#?}",
461                            self.stmt_info, target
462                        );
463                    };
464                    let (index, _) = self.load_data.inputs.insert_full(node_id);
465                    batch_load_index.set(insert_row);
466                    input.set(Some(index));
467                }
468                hir::Arg::Ref {
469                    stmt_id: target_id,
470                    data_load_input,
471                    batch_load_index,
472                    ..
473                } => {
474                    // refs can be duplicated in the same statement
475                    if data_load_input.get().is_some() {
476                        return;
477                    }
478
479                    let target_stmt_info = &self.planner.hir[target_id];
480                    let back_ref = &target_stmt_info.back_refs[&self.stmt_id];
481
482                    // TODO: should we just use the data_load node ID?
483                    let node_id = back_ref.node_id.get().unwrap();
484
485                    let (index, _) = self.load_data.inputs.insert_full(node_id);
486                    data_load_input.set(Some(index));
487
488                    // If the target statement is a query, then we are in a batch-load scenario.
489                    if target_stmt_info.stmt().is_query() {
490                        debug_assert!(insert_row.is_none());
491
492                        let (batch_load_table_ref_index, _) =
493                            self.load_data.batch_load_args.insert_full(index);
494                        batch_load_index.set(Some(batch_load_table_ref_index));
495                    } else if let Some(row) = insert_row {
496                        debug_assert!(target_stmt_info.stmt().is_insert());
497                        // batch_load_index may already be set during lowering
498                        // (when the parent INSERT's row index was captured via
499                        // scope_statement). In that case, the lowering value is
500                        // the correct parent row index; don't overwrite it.
501                        if batch_load_index.get().is_none() {
502                            batch_load_index.set(Some(row));
503                        }
504                    } else {
505                        debug_assert!(
506                            batch_load_index.get().is_some(),
507                            "stmt={:#?}; target={:#?}; batch_load_index={:#?}",
508                            self.stmt_info,
509                            target_stmt_info,
510                            batch_load_index.get()
511                        );
512                    }
513                }
514            }
515        }
516    }

526    fn rewrite_stmt_for_batch_load(&mut self, stmt: &mut stmt::Statement) {
527        if self.planner.engine.capability().sql {
528            self.rewrite_stmt_query_for_batch_load_sql(stmt);
529        } else {
530            self.rewrite_stmt_query_for_batch_load_nosql(stmt);
531        }

534    fn rewrite_stmt_query_for_batch_load_sql(&mut self, stmt: &mut stmt::Statement) {
535        // If targeting SQL, leverage the SQL query engine to handle most of the rewrite details.
536        let mut filter = stmt
537            .filter_mut()
538            .map(|filter| filter.take())
539            .unwrap_or_default();
540
541        visit_mut::for_each_expr_mut(&mut filter, |expr| {
542            match expr {
543                stmt::Expr::Reference(stmt::ExprReference::Column(expr_column)) => {
544                    debug_assert_eq!(0, expr_column.nesting);
545                    // We need to up the nesting to reflect that the filter is moved
546                    // one level deeper.
547                    expr_column.nesting += 1;
548                }
549                stmt::Expr::Arg(expr_arg) => {
550                    let hir::Arg::Ref {
551                        stmt_id: target_id,
552                        target_expr_ref,
553                        batch_load_index: batch_load_table_ref_index,
554                        ..
555                    } = &self.stmt_info.args[expr_arg.position]
556                    else {
557                        todo!()
558                    };
559
560                    let back_ref = &self.planner.hir[target_id].back_refs[&self.stmt_id];
561                    let column = back_ref.exprs.get_index_of(target_expr_ref).unwrap();
562
563                    // Rewrite reference the new `FROM`.
564                    *expr = stmt::Expr::column(stmt::ExprColumn {
565                        nesting: 0,
566                        table: batch_load_table_ref_index.get().unwrap(),
567                        column,
568                    });
569                }
570                _ => {}
571            }
572        });
573
574        let tables: Vec<stmt::TableRef> = self
575            .load_data
576            .batch_load_args
577            .iter()
578            .map(|position| stmt::TableRef::Arg(stmt::ExprArg::new(*position)))
579            .collect();
580
581        assert!(tables.len() <= 1, "TODO: handle more complicated cases");
582
583        let sub_query = stmt::Select {
584            returning: stmt::Returning::Project(stmt::Expr::record([1])),
585            source: stmt::Source::Table(stmt::SourceTable {
586                tables,
587                from: vec![stmt::TableWithJoins {
588                    relation: stmt::TableFactor::Table(stmt::SourceTableId(0)),
589                    joins: vec![],
590                }],
591            }),
592            filter,
593        };
594
595        stmt.filter_mut_unwrap().set(stmt::Expr::exists(sub_query));
596    }

598    fn rewrite_stmt_query_for_batch_load_nosql(&mut self, stmt: &mut stmt::Statement) {
599        let mut filter = stmt.filter_expr_mut();
600        visit_mut::for_each_expr_mut(&mut filter, |expr| match expr {
601            stmt::Expr::Reference(stmt::ExprReference::Column(expr_column)) => {
602                debug_assert_eq!(0, expr_column.nesting);
603            }
604            stmt::Expr::Arg(expr_arg) => {
605                let hir::Arg::Ref {
606                    stmt_id: target_id,
607                    target_expr_ref,
608                    ..
609                } = &self.stmt_info.args[expr_arg.position]
610                else {
611                    todo!()
612                };
613
614                let back_ref = &self.planner.hir[target_id].back_refs[&self.stmt_id];
615                let column = back_ref.exprs.get_index_of(target_expr_ref).unwrap();
616
617                *expr = stmt::Expr::arg(column);
618            }
619            _ => {}
620        });
621
622        assert!(
623            self.load_data.batch_load_args.len() == 1,
624            "TODO: handle more complicated cases"
625        );
626        let input = self.load_data.batch_load_args[0];
627
628        if let Some(filter) = filter {
629            let expr = filter.take();
630            *filter = stmt::Expr::any(stmt::Expr::map(stmt::Expr::arg(input), expr));
631        }
632    }

644    fn rewrite_stmt_update_arg_dependencies(&mut self, stmt: &mut stmt::Update) {
645        for (_, assignment) in stmt.assignments.iter_mut() {
646            let expr = match assignment {
647                stmt::Assignment::Set(expr)
648                | stmt::Assignment::Insert(expr)
649                | stmt::Assignment::Remove(expr)
650                | stmt::Assignment::Append(expr)
651                | stmt::Assignment::RemoveAt(expr) => expr,
652                stmt::Assignment::Pop => continue,
653                stmt::Assignment::Batch(_) => {
654                    todo!("batch assignments in arg dependency rewriting")
655                }
656            };
657            self.rewrite_arg_dependencies(expr);
658        }
659    }

661    fn rewrite_arg_dependencies(&mut self, expr: &mut stmt::Expr) {
662        visit_mut::for_each_expr_mut(expr, |expr| {
663            if let stmt::Expr::Arg(expr_arg) = expr {
664                match &self.stmt_info.args[expr_arg.position] {
665                    hir::Arg::Ref {
666                        stmt_id: target_id,
667                        target_expr_ref,
668                        data_load_input,
669                        batch_load_index,
670                        ..
671                    } => {
672                        debug_assert!(!self.load_data.inputs.is_empty(), "{:#?}", self.load_data);
673
674                        // TODO: this work seems to be duplicated in the returning as well.
675                        let back_ref = &self.planner.hir[target_id].back_refs[&self.stmt_id];
676                        let column = back_ref.exprs.get_index_of(target_expr_ref).unwrap();
677
678                        *expr = stmt::Expr::arg_project(
679                            data_load_input.get().unwrap(),
680                            [batch_load_index.get().unwrap(), column],
681                        );
682                    }
683                    hir::Arg::Sub { input, .. } => {
684                        debug_assert!(
685                            !self.load_data.inputs.is_empty(),
686                            "{:#?} | is this needed?",
687                            self.load_data
688                        );
689                        *expr = stmt::Expr::arg(input.get().unwrap());
690                    }
691                }
692            }
693        });
694    }

698    fn plan_data_loading(
699        &mut self,
700        stmt: stmt::Statement,
701        returning: &mut Returning,
702    ) -> Result<mir::NodeId> {
703        // COUNT(*) is SQL-only
704        if self.load_data.select_items.contains(&SelectItem::CountStar)
705            && !self.planner.engine.capability().sql
706        {
707            return Err(toasty_core::Error::unsupported_feature(
708                "count() queries are only supported with SQL drivers",
709            ));
710        }
711
712        if let Some(node_id) = self.plan_const_or_empty_statement(&stmt, returning) {
713            debug_assert!(
714                stmt.is_query() || stmt.assignments().map(|a| a.is_empty()).unwrap_or(false),
715                "planned a mutable statement as const; stmt={:#?}",
716                stmt
717            );
718            Ok(node_id)
719        } else if self.planner.engine.capability().sql || stmt.is_insert() {
720            self.plan_data_loading_sql(stmt)
721        } else {
722            self.plan_data_loading_nosql(stmt)
723        }
724    }

765    fn plan_data_loading_sql(&mut self, mut stmt: stmt::Statement) -> Result<mir::NodeId> {
766        let const_returning = self.extract_insert_returning_as_const(&stmt);
767
768        // Phase 1: Detect pagination and add ORDER BY columns to load_data
769        let pagination_info = self.plan_pagination_sql(&stmt)?;
770
771        // Set returning clause with all columns (including added ORDER BY columns)
772        if !self.load_data.select_items.is_empty() {
773            stmt.set_returning_project(stmt::Expr::record(
774                self.load_data
775                    .select_items
776                    .iter()
777                    .map(|item| item.to_expr()),
778            ));
779        }
780
781        let input_args: Vec<_> = self
782            .load_data
783            .inputs
784            .iter()
785            .map(|input| self.planner.mir.ty(*input).clone())
786            .collect();
787
788        // Infer type after adding all columns
789        let ty = self.planner.engine.infer_ty(&stmt, &input_args[..]);
790
791        // Phase 2: Build extract_cursor function using the inferred type
792        let pagination_config = pagination_info.map(|info| self.build_extract_cursor(info, &ty));
793
794        let node = if stmt.condition().is_some() {
795            if let stmt::Statement::Update(stmt) = stmt {
796                assert!(stmt.returning.is_none(), "TODO: stmt={stmt:#?}");
797
798                if self.planner.engine.capability().cte_with_update {
799                    mir::Operation::ExecStatement(Box::new(
800                        self.plan_conditional_sql_query_as_cte(stmt, ty),
801                    ))
802                } else {
803                    mir::Operation::ReadModifyWrite(Box::new(
804                        self.plan_conditional_sql_query_as_rmw(stmt, ty),
805                    ))
806                }
807            } else {
808                todo!("stmt={stmt:#?}");
809            }
810        } else {
811            debug_assert!(
812                stmt.returning()
813                    .and_then(|returning| returning.as_project())
814                    .map(|expr| expr.is_record())
815                    .unwrap_or(true),
816                "stmt={stmt:#?}"
817            );
818            // With SQL capability, we can just punt the details of execution to
819            // the database's query planner.
820            mir::Operation::ExecStatement(Box::new(mir::ExecStatement {
821                inputs: mem::take(&mut self.load_data.inputs),
822                stmt,
823                ty,
824                conditional_update_with_no_returning: false,
825                pagination: pagination_config.clone(),
826            }))
827        };
828
829        // With SQL capability, we can just punt the details of execution to
830        // the database's query planner.
831        let mut exec_statement_node = self.insert_mir_with_deps(node);
832
833        if let Some((const_value, const_ty)) = const_returning {
834            exec_statement_node = self.planner.mir.insert_with_deps(
835                mir::Const {
836                    value: const_value,
837                    ty: const_ty,
838                },
839                [exec_statement_node],
840            );
841        }
842
843        Ok(exec_statement_node)
844    }

846    fn extract_insert_returning_as_const(
847        &mut self,
848        stmt: &stmt::Statement,
849    ) -> Option<(stmt::Value, stmt::Type)> {
850        let stmt::Statement::Insert(insert) = stmt else {
851            return None;
852        };
853
854        if self.load_data.select_items.is_empty() {
855            return None;
856        }
857
858        let target = insert.target.as_table_unwrap();
859        let values = insert.source.body.as_values()?;
860
861        let mut indices = vec![];
862
863        for select_item in &self.load_data.select_items {
864            let expr_ref = select_item.as_expr_reference_unwrap();
865            let expr_col = expr_ref.as_expr_column_unwrap();
866            debug_assert!(expr_col.nesting == 0, "expr_column={expr_col:#?}");
867
868            let Some(index) = target
869                .columns
870                .iter()
871                .enumerate()
872                .find(|(_, column_id)| column_id.index == expr_col.column)
873                .map(|(index, _)| index)
874            else {
875                todo!("insert returning referencing parent statement");
876                // return None;
877            };
878
879            indices.push(index);
880        }
881
882        // Now extract the values for each row
883        let mut result = Vec::with_capacity(values.rows.len());
884
885        for row in &values.rows {
886            // Build a record with only the requested fields
887            let mut fields = Vec::with_capacity(indices.len());
888
889            for &index in &indices {
890                // Try to evaluate the expression to a constant value
891                let value = row.entry(index)?.eval_const().ok()?;
892                fields.push(value);
893            }
894
895            result.push(stmt::Value::record_from_vec(fields));
896        }
897
898        let ty = self
899            .load_data
900            .select_items
901            .infer_record_list_ty(&self.planner.engine.expr_cx_for(stmt));
902
903        Some((stmt::Value::List(result), ty))
904    }

906    fn plan_pagination_sql(&mut self, stmt: &stmt::Statement) -> Result<Option<PaginationInfo>> {
907        let stmt::Statement::Query(query) = stmt else {
908            return Ok(None);
909        };
910
911        // Only cursor-based limits trigger pagination planning
912        let Some(limit) = query.limit.as_ref() else {
913            return Ok(None);
914        };
915        let stmt::Limit::Cursor(cursor) = limit else {
916            return Ok(None);
917        };
918
919        // SQL cursor pagination requires ORDER BY to produce a deterministic cursor.
920        let order_by = query.order_by.as_ref().ok_or_else(|| {
921            toasty_core::Error::unsupported_feature(
922                "cursor-based pagination requires an ORDER BY clause on SQL drivers",
923            )
924        })?;
925
926        // Extract page_size
927        let page_size = match &cursor.page_size {
928            stmt::Expr::Value(stmt::Value::I64(n)) => *n,
929            _ => return Ok(None),
930        };
931
932        // Add ORDER BY columns to load_data so they're available for cursor extraction
933        let mut cursor_column_indices = Vec::new();
934
935        for order_expr in &order_by.exprs {
936            // Try to convert the ORDER BY expression to an ExprReference
937            if let Some(expr_ref) = order_expr.expr.as_expr_reference().copied() {
938                // Add to load_data if not already present
939                let (index, _) = self
940                    .load_data
941                    .select_items
942                    .insert_full(SelectItem::from(expr_ref));
943                cursor_column_indices.push(index);
944            } else {
945                // Complex expression in ORDER BY - can't handle yet
946                return Ok(None);
947            }
948        }
949
950        Ok(Some(PaginationInfo {
951            page_size,
952            cursor_column_indices,
953        }))
954    }

956    /// Builds extract_cursor function using the inferred row type.
957    /// Called after type inference in plan_data_loading_sql.
958    fn build_extract_cursor(
959        &self,
960        info: PaginationInfo,
961        ty: &stmt::Type,
962    ) -> exec::PaginationConfig {
963        // Extract row type from List<Row>
964        let row_ty = if let stmt::Type::List(item_ty) = ty {
965            (**item_ty).clone()
966        } else {
967            stmt::Type::Unit
968        };
969
970        // Build extract_cursor expression: projects ORDER BY column positions from the row
971        let extract_cursor = stmt::Expr::record(
972            info.cursor_column_indices
973                .into_iter()
974                .map(|index| stmt::Expr::arg_project(0, [index])),
975        );
976
977        // Build eval::Func with the actual row type as input
978        let extract_cursor_func = eval::Func::from_stmt(extract_cursor, vec![row_ty]);
979
980        exec::PaginationConfig {
981            page_size: info.page_size,
982            extract_cursor: Some(extract_cursor_func),
983        }
984    }

986    fn plan_conditional_sql_query_as_cte(
987        &mut self,
988        stmt: stmt::Update,
989        ty: stmt::Type,
990    ) -> mir::ExecStatement {
991        let Some(condition) = stmt.condition.expr else {
992            panic!("conditional update without condition");
993        };
994
995        let Some(filter) = stmt.filter.expr else {
996            panic!("conditional update without filter");
997        };
998
999        let stmt::UpdateTarget::Table(target) = stmt.target.clone() else {
1000            panic!("conditional update without table");
1001        };
1002
1003        let mut ctes = vec![];
1004
1005        // Select from update table without the update condition.
1006        ctes.push(stmt::Cte {
1007            query: stmt::Query::builder(target)
1008                .filter(filter.clone())
1009                .returning_project(stmt::Expr::record_from_vec(vec![
1010                    stmt::Expr::count_star(),
1011                    stmt::FuncCount {
1012                        arg: None,
1013                        filter: Some(Box::new(condition)),
1014                    }
1015                    .into(),
1016                ]))
1017                .build(),
1018        });
1019
1020        let returning_len = match &stmt.returning {
1021            Some(stmt::Returning::Project(expr)) => {
1022                let stmt::Expr::Record(expr_record) = expr else {
1023                    panic!("returning must be a record");
1024                };
1025
1026                expr_record.fields.len()
1027            }
1028            Some(_) => todo!(),
1029            None => 0,
1030        };
1031
1032        // The update statement. The update condition is expressed using the select above
1033        ctes.push(stmt::Cte {
1034            query: stmt::Query::new(stmt::Update {
1035                target: stmt.target,
1036                assignments: stmt.assignments,
1037                filter: stmt::Filter::new(stmt::Expr::and(
1038                    filter,
1039                    // SELECT found.count(*) = found.count(CONDITION) FROM found
1040                    stmt::Expr::stmt(stmt::Select {
1041                        source: stmt::TableRef::Cte {
1042                            nesting: 2,
1043                            index: 0,
1044                        }
1045                        .into(),
1046                        filter: true.into(),
1047                        returning: stmt::Returning::Project(stmt::Expr::record_from_vec(vec![
1048                            stmt::Expr::eq(
1049                                stmt::ExprColumn {
1050                                    nesting: 0,
1051                                    table: 0,
1052                                    column: 0,
1053                                },
1054                                stmt::ExprColumn {
1055                                    nesting: 0,
1056                                    table: 0,
1057                                    column: 1,
1058                                },
1059                            ),
1060                        ])),
1061                    }),
1062                )),
1063                condition: Condition::default(),
1064                returning: Some(
1065                    stmt.returning
1066                        // TODO: hax
1067                        .unwrap_or_else(|| {
1068                            stmt::Returning::Project(stmt::Expr::record_from_vec(vec![
1069                                stmt::Expr::from("hello"),
1070                            ]))
1071                        }),
1072                ),
1073            }),
1074        });
1075
1076        let mut columns = vec![
1077            stmt::Expr::column(stmt::ExprColumn {
1078                nesting: 0,
1079                table: 0,
1080                column: 0,
1081            }),
1082            stmt::Expr::column(stmt::ExprColumn {
1083                nesting: 0,
1084                table: 0,
1085                column: 1,
1086            }),
1087        ];
1088
1089        for i in 0..returning_len {
1090            columns.push(stmt::Expr::column(stmt::ExprColumn {
1091                nesting: 0,
1092                table: 1,
1093                column: i,
1094            }));
1095        }
1096
1097        let stmt = stmt::Query::builder(stmt::Select {
1098            source: stmt::Source::table_with_joins(
1099                vec![
1100                    stmt::TableRef::Cte {
1101                        nesting: 0,
1102                        index: 0,
1103                    },
1104                    stmt::TableRef::Cte {
1105                        nesting: 0,
1106                        index: 1,
1107                    },
1108                ],
1109                stmt::TableWithJoins {
1110                    relation: stmt::TableFactor::Table(stmt::SourceTableId(0)),
1111                    joins: vec![stmt::Join {
1112                        table: stmt::SourceTableId(1),
1113                        constraint: stmt::JoinOp::Left(stmt::Expr::from(true)),
1114                    }],
1115                },
1116            ),
1117            filter: stmt::Filter::new(true),
1118            returning: stmt::Returning::Project(stmt::Expr::record_from_vec(columns)),
1119        })
1120        .with(ctes)
1121        .build()
1122        .into();
1123
1124        mir::ExecStatement {
1125            inputs: mem::take(&mut self.load_data.inputs),
1126            stmt,
1127            ty,
1128            conditional_update_with_no_returning: true,
1129            pagination: None,
1130        }
1131    }

1191    fn plan_data_loading_nosql(&mut self, stmt: stmt::Statement) -> Result<mir::NodeId> {
1192        if stmt.is_insert() {
1193            debug_assert!(self.load_data.select_items.is_empty());
1194        }
1195
1196        // Without SQL capability, we have to plan the execution of the
1197        // statement based on available indices.
1198        let index_plan_opt = self.planner.engine.plan_index_path(&stmt)?;
1199
1200        if let Some(mut index_plan) = index_plan_opt {
1201            // prepare_post_filter may insert additional columns (e.g. deferred
1202            // fields used only in a post-filter) into select_items, so the
1203            // record type must be computed AFTER this call.
1204            let post_filter = self.prepare_post_filter(&stmt, &mut index_plan);
1205
1206            let ty = self.infer_nosql_record_ty(&stmt);
1207
1208            let node_id = if index_plan.index.primary_key {
1209                self.plan_primary_key_execution(stmt, &mut index_plan, &ty)
1210            } else {
1211                self.plan_secondary_index_execution(stmt, &mut index_plan, &ty)
1212            };
1213
1214            Ok(self.apply_post_filter(node_id, post_filter, ty))
1215        } else {
1216            // No index covers the filter — emit a full-table scan.
1217            let ty = self.infer_nosql_record_ty(&stmt);
1218            self.plan_scan_execution(stmt, ty)
1219        }
1220    }

1226                // an empty record for every matching row.
1227                stmt::Type::list(stmt::Type::Record(vec![]))

1229                stmt::Type::Unit
1230            }
1231        } else {

1233                .select_items
1234                .infer_record_list_ty(&self.planner.engine.expr_cx_for(stmt))

1238    fn plan_scan_execution(
1239        &mut self,
1240        stmt: stmt::Statement,
1241        ty: stmt::Type,
1242    ) -> Result<mir::NodeId> {
1243        let input = if self.load_data.inputs.is_empty() {
1244            None
1245        } else if self.load_data.inputs.len() == 1 {
1246            Some(self.load_data.inputs[0])
1247        } else {
1248            todo!("scan with multiple inputs")
1249        };
1250
1251        let cx = stmt::ExprContext::new(&*self.planner.engine.schema);
1252        let cx = cx.scope(&stmt);
1253        let stmt::ExprTarget::Table(table) = cx.target() else {
1254            return Err(toasty_core::Error::unsupported_feature(
1255                "scan: expected table target",
1256            ));
1257        };
1258        let table_id = table.id;
1259
1260        // Reject ORDER BY on drivers whose scan operation returns items in an
1261        // unspecified order (e.g. DynamoDB Scan has no server-side sort).
1262        if let Some(query) = stmt.as_query()
1263            && query.order_by.is_some()
1264            && !self.planner.engine.capability().scan_supports_sort
1265        {
1266            return Err(toasty_core::Error::unsupported_feature(
1267                "ORDER BY is not supported on full-table scans for this database. \
1268                     Consider adding an index on the sort field or removing the ORDER BY clause.",
1269            ));
1270        }
1271
1272        let row_filter = {
1273            let f = stmt.filter_expr_unwrap();
1274            if f.is_true() { None } else { Some(f.clone()) }
1275        };
1276
1277        let limit = extract_pagination(&stmt);
1278
1279        Ok(self.insert_mir_with_deps(mir::Scan {
1280            input,
1281            table: table_id,
1282            columns: self.load_data.select_items.extract_expr_references(),
1283            row_filter,
1284            limit,
1285            ty,
1286        }))
1287    }

1289    fn plan_primary_key_execution(
1290        &mut self,
1291        stmt: stmt::Statement,
1292        index_plan: &mut index::IndexPlan,
1293        ty: &stmt::Type,
1294    ) -> mir::NodeId {
1295        if let Some(mut key_expr) = index_plan.key_values.take() {
1296            let (args, _input_nodes) = self.rewrite_expr_for_mir(&mut key_expr);
1297            let key_ty =
1298                stmt::Type::list(self.planner.engine.index_key_record_ty(index_plan.index));
1299            let keys = eval::Func::from_stmt_typed(key_expr, args, key_ty);
1300            let get_by_key_input = self.build_get_by_key_input(keys, self.index_key_ty(index_plan));
1301
1302            self.build_key_operation(&stmt, index_plan, get_by_key_input, ty)
1303        } else {
1304            let input = if self.load_data.inputs.is_empty() {
1305                None
1306            } else if self.load_data.inputs.len() == 1 {
1307                Some(self.load_data.inputs[0])
1308            } else {
1309                todo!()
1310            };
1311
1312            if stmt.is_query() {
1313                let limit = extract_pagination(&stmt);
1314                let order = extract_query_pk_order(&stmt);
1315
1316                // For queries, stream all matching records with the requested columns.
1317                self.insert_mir_with_deps(mir::QueryPk {
1318                    input,
1319                    table: index_plan.table_id(),
1320                    index: None, // Querying primary key
1321                    columns: self.load_data.select_items.extract_expr_references(),
1322                    pk_filter: index_plan.index_filter.take(),
1323                    row_filter: index_plan.result_filter.take(),
1324                    ty: ty.clone(),
1325                    limit,
1326                    order,
1327                })
1328            } else {
1329                // For mutations (UPDATE/DELETE) with a partial primary-key filter,
1330                // first collect the full primary keys of all matching records via
1331                // QueryPk, then apply the mutation to each key. The index key columns
1332                // were pre-populated into load_data.select_items in plan_data_loading_nosql.
1333                let index_key_ty = self.index_key_ty(index_plan);
1334
1335                let mut columns = self.load_data.select_items.extract_expr_references();
1336                assert!(columns.is_empty());
1337
1338                for index_col in &index_plan.index.columns {
1339                    columns.insert(stmt::ExprReference::Column(stmt::ExprColumn {
1340                        nesting: 0,
1341                        table: 0,
1342                        column: index_col.column.index,
1343                    }));
1344                }
1345
1346                let query_pk_node = self.insert_mir_with_deps(mir::QueryPk {
1347                    input,
1348                    table: index_plan.table_id(),
1349                    index: None, // Querying primary key
1350                    columns,
1351                    pk_filter: index_plan.index_filter.take(),
1352                    row_filter: index_plan.result_filter.take(),
1353                    ty: index_key_ty,
1354                    limit: None,
1355                    order: None,
1356                });
1357
1358                self.build_key_operation(&stmt, index_plan, query_pk_node, ty)
1359            }
1360        }
1361    }

1363    fn plan_secondary_index_execution(
1364        &mut self,
1365        stmt: stmt::Statement,
1366        index_plan: &mut index::IndexPlan,
1367        ty: &stmt::Type,
1368    ) -> mir::NodeId {
1369        let inputs = mem::take(&mut self.load_data.inputs);
1370        assert!(index_plan.post_filter.is_none(), "TODO");
1371        assert!(inputs.len() <= 1, "TODO: inputs={:#?}", inputs);
1372
1373        // For queries on NON-UNIQUE indexes, use QueryPk optimization
1374        // Non-unique indexes are created as DynamoDB GSIs with ProjectionType::All,
1375        // meaning they contain all table columns and can return full records.
1376        // Unique indexes are created as separate tables and don't support this optimization.
1377        // This method is already on the nosql path, so we can assume the behavior of unique indexes.
1378        // To support DDB's other projection types, Index will need to track projected columns, not
1379        // just the columns that are part of the index key.
1380        if stmt.is_query() && !index_plan.index.unique {
1381            let input = if inputs.is_empty() {
1382                None
1383            } else {
1384                Some(inputs[0])
1385            };
1386
1387            let limit = extract_pagination(&stmt);
1388            let order = extract_query_pk_order(&stmt);
1389
1390            // Use QueryPk with index to query the secondary index and return full records
1391            // This eliminates the N+1 pattern of FindPkByIndex + GetByKey
1392            return self.insert_mir_with_deps(mir::QueryPk {
1393                input,
1394                table: index_plan.index.on,
1395                index: Some(index_plan.index.id), // Query the secondary index
1396                columns: self.load_data.select_items.extract_expr_references(), // Return full records
1397                pk_filter: index_plan.index_filter.take(),
1398                row_filter: index_plan.result_filter.take(),
1399                ty: ty.clone(), // Full record type, not just PKs
1400                limit,
1401                order,
1402            });
1403        }
1404
1405        // For mutations, unique indexes, or other cases, use FindPkByIndex + GetByKey
1406        // - Mutations only need primary keys, not full records
1407        // - Unique indexes don't have full column projections in DynamoDB
1408        let index_key_ty = self.index_key_ty(index_plan);
1409
1410        let get_by_key_input = self.insert_mir_with_deps(mir::FindPkByIndex {
1411            inputs,
1412            table: index_plan.index.on,
1413            index: index_plan.index.id,
1414            filter: index_plan.index_filter.take(),
1415            ty: index_key_ty,
1416        });
1417
1418        self.build_key_operation(&stmt, index_plan, get_by_key_input, ty)
1419    }

1421    fn prepare_post_filter(
1422        &mut self,
1423        stmt: &stmt::Statement,
1424        index_plan: &mut index::IndexPlan,
1425    ) -> Option<stmt::Expr> {
1426        let mut post_filter = index_plan.post_filter.clone();
1427
1428        // If fetching rows using GetByKey, some databases do not support
1429        // applying additional filters to the rows before returning results.
1430        // In this case, the result_filter needs to be applied in-memory.
1431        if stmt.is_query()
1432            && (index_plan.has_pk_keys || !index_plan.index.primary_key)
1433            && let Some(result_filter) = index_plan.result_filter.take()
1434        {
1435            post_filter = Some(match post_filter {
1436                Some(post_filter) => stmt::Expr::and(result_filter, post_filter),
1437                None => result_filter,
1438            });
1439        }
1440
1441        debug_assert!(
1442            post_filter.is_none() || stmt.is_query(),
1443            "stmt={:#?}; post_filter={post_filter:#?}",
1444            stmt
1445        );
1446
1447        // Make sure we are including columns needed to apply the post filter
1448        if let Some(post_filter) = &mut post_filter {
1449            visit_mut::for_each_expr_mut(post_filter, |expr| match expr {
1450                stmt::Expr::Reference(expr_reference) => {
1451                    let (index, _) = self
1452                        .load_data
1453                        .select_items
1454                        .insert_full((*expr_reference).into());
1455                    *expr = stmt::Expr::arg_project(0, [index]);
1456                }
1457                stmt::Expr::Arg(_) => todo!("expr={expr:#?}"),
1458                _ => {}
1459            });
1460        }
1461
1462        post_filter
1463    }

1505    fn build_key_operation(
1506        &mut self,
1507        stmt: &stmt::Statement,
1508        index_plan: &mut index::IndexPlan,
1509        get_by_key_input: mir::NodeId,
1510        ty: &stmt::Type,
1511    ) -> mir::NodeId {
1512        match stmt {
1513            stmt::Statement::Query(_) => {
1514                debug_assert!(ty.is_list(), "ty={ty:#?}");
1515                self.insert_mir_with_deps(mir::GetByKey {
1516                    input: get_by_key_input,
1517                    table: index_plan.table_id(),
1518                    columns: self.load_data.select_items.extract_expr_references(),
1519                    ty: ty.clone(),
1520                })
1521            }
1522            stmt::Statement::Delete(delete_stmt) => self.insert_mir_with_deps(mir::DeleteByKey {
1523                input: get_by_key_input,
1524                table: index_plan.table_id(),
1525                filter: index_plan.result_filter.take(),
1526                condition: delete_stmt.condition.expr.clone(),
1527                ty: stmt::Type::Unit,
1528            }),
1529            stmt::Statement::Update(update_stmt) => {
1530                // If there is a pre-filter, wrap the key input in a Guard
1531                // node that produces an empty list when the guard is false,
1532                // causing UpdateByKey to naturally no-op.
1533                let guarded_input = self.apply_guard(get_by_key_input, index_plan);
1534
1535                self.insert_mir_with_deps(mir::UpdateByKey {
1536                    input: guarded_input,
1537                    table: index_plan.table_id(),
1538                    assignments: update_stmt.assignments.clone(),
1539                    filter: index_plan.result_filter.take(),
1540                    condition: update_stmt.condition.expr.clone(),
1541                    ty: ty.clone(),
1542                })
1543            }
1544            _ => todo!("stmt={stmt:#?}"),
1545        }
1546    }

1548    /// If the index plan has a pre-filter, insert a `Guard` MIR node that
1549    /// wraps the given input. When the guard expression evaluates to false,
1550    /// the Guard produces an empty list, causing the downstream operation to
1551    /// see no keys and become a no-op. Returns the (possibly wrapped) input
1552    /// node ID.
1553    fn apply_guard(
1554        &mut self,
1555        input: mir::NodeId,
1556        index_plan: &mut index::IndexPlan,
1557    ) -> mir::NodeId {
1558        let Some(mut pre_filter_expr) = index_plan.pre_filter.take() else {
1559            return input;
1560        };
1561
1562        let (args, guard_inputs) = self.rewrite_expr_for_mir(&mut pre_filter_expr);
1563        let guard = eval::Func::from_stmt(pre_filter_expr, args);
1564        let ty = self.planner.mir[input].ty().clone();
1565
1566        self.planner.mir.insert(mir::Guard {
1567            input,
1568            guard_inputs,
1569            guard,
1570            ty,
1571        })
1572    }

1574    /// Rewrite a statement-level expression for use in a MIR node.
1575    ///
1576    /// Statement-level expressions contain `Arg(n)` where `n` is a position in
1577    /// `stmt_info.args` (the HIR arg list). Each HIR arg maps to an entry in
1578    /// `load_data.inputs` via `hir::Arg::Sub { input, .. }`.
1579    ///
1580    /// MIR nodes have their own compact input lists. This method:
1581    /// 1. Resolves each HIR arg to its `load_data.inputs` node ID
1582    /// 2. Assigns a new compact position (index into the returned inputs)
1583    /// 3. Rewrites the `Arg` position in the expression
1584    ///
1585    /// Returns `(arg_types, input_node_ids)` for constructing the MIR node.
1586    fn rewrite_expr_for_mir(
1587        &self,
1588        expr: &mut stmt::Expr,
1589    ) -> (Vec<stmt::Type>, IndexSet<mir::NodeId>) {
1590        let mut arg_map: IndexMap<usize, (stmt::Type, mir::NodeId)> = IndexMap::new();
1591
1592        visit_mut::for_each_expr_mut(expr, |expr| {
1593            if let stmt::Expr::Arg(expr_arg) = expr {
1594                let hir_pos = expr_arg.position;
1595                let new_pos = match arg_map.get_index_of(&hir_pos) {
1596                    Some(idx) => idx,
1597                    None => {
1598                        // Resolve the HIR arg to its load_data input
1599                        let input_idx = match &self.stmt_info.args[hir_pos] {
1600                            hir::Arg::Sub { input, .. } => input.get().unwrap(),
1601                            _ => todo!("rewrite_expr_for_mir with non-Sub arg"),
1602                        };
1603                        let node_id = self.load_data.inputs[input_idx];
1604                        let ty = self.planner.mir[node_id].ty().clone();
1605                        let (idx, _) = arg_map.insert_full(hir_pos, (ty, node_id));
1606                        idx
1607                    }
1608                };
1609                expr_arg.position = new_pos;
1610            }
1611        });
1612
1613        let mut types = Vec::with_capacity(arg_map.len());
1614        let mut nodes = IndexSet::with_capacity(arg_map.len());
1615        for (_, (ty, node_id)) in arg_map {
1616            types.push(ty);
1617            nodes.insert(node_id);
1618        }
1619        (types, nodes)
1620    }

1671    fn plan_output_node(
1672        &mut self,
1673        data_load_node_id: mir::NodeId,
1674        returning: ReturningInfo,
1675    ) -> mir::NodeId {
1676        // First check for nested merge
1677        if let Some(node_id) = self.planner.plan_nested_merge(self.stmt_id) {
1678            return node_id;
1679        }
1680
1681        let returning_arg_tys = returning
1682            .inputs
1683            .iter()
1684            .map(|input| self.planner.mir[input].ty().clone())
1685            .collect();
1686
1687        // Then handle returning clause
1688        if let Some(clause) = returning.clause {
1689            match clause {
1690                stmt::Returning::Expr(expr) => {
1691                    // Value variant contains a constant expression that can be evaluated
1692                    if let Ok(value) = expr.eval_const() {
1693                        let ty = value.infer_ty();
1694
1695                        self.planner
1696                            .mir
1697                            .insert_with_deps(mir::Const { value, ty }, [data_load_node_id])
1698                    } else {
1699                        let eval = eval::Func::from_stmt(expr, returning_arg_tys);
1700
1701                        let node_id = self.insert_mir_with_deps(mir::Eval {
1702                            inputs: returning.inputs,
1703                            eval,
1704                            metadata: None,
1705                        });
1706
1707                        if !self.stmt().is_query() {
1708                            self.planner.mir[node_id].deps.insert(data_load_node_id);
1709                        }
1710
1711                        node_id
1712                    }
1713                }
1714                stmt::Returning::Project(projection) => {
1715                    if let Some(position) = returning.inputs.get_index_of(&data_load_node_id) {
1716                        self.insert_mir_with_deps(mir::Eval {
1717                            inputs: returning.inputs,
1718                            eval: eval::Func::from_stmt(
1719                                stmt::Expr::map(stmt::Expr::arg(position), projection),
1720                                returning_arg_tys,
1721                            ),
1722                            metadata: Some(position),
1723                        })
1724                    } else {
1725                        // TODO: figure out how to handle repeating a number of times vs. projecting results
1726                        let projection = eval::Func::from_stmt(projection, vec![]);
1727                        let ty = stmt::Type::list(projection.ret.clone());
1728
1729                        let node = mir::Project {
1730                            input: data_load_node_id,
1731                            projection,
1732                            ty,
1733                        };
1734
1735                        // Plan the final projection to handle the returning clause.
1736                        self.insert_mir_with_deps(node)
1737                    }
1738                }
1739                returning => panic!("unexpected `stmt::Returning` kind; returning={returning:#?}"),
1740            }
1741        } else {
1742            self.apply_dependencies_to_node(data_load_node_id);
1743            data_load_node_id
1744        }
1745    }

1796/// Extract limit/pagination bounds from a query statement for use with
1797/// `QueryPk` on NoSQL drivers. Returns `None` when the statement has no limit
1798/// clause.
1799///
1800/// Assumes `page_size`, `limit`, and `offset` fields are `Value::I64` literals.
1801/// Builders normalize to `I64`, and `verify::verify_limit_is_integer_literal`
1802/// enforces this invariant on the AST — so any other shape reaching here is a
1803/// bug upstream.
1804fn extract_pagination(stmt: &stmt::Statement) -> Option<Pagination> {
1805    let query = stmt.as_query()?;
1806    match query.limit.as_ref()? {
1807        stmt::Limit::Cursor(c) => {
1808            let page_size = as_i64_literal(&c.page_size);
1809            let after = c.after.as_ref().and_then(|e| match e {
1810                stmt::Expr::Value(v) => Some(v.clone()),
1811                _ => None,
1812            });
1813            Some(Pagination::Cursor { page_size, after })
1814        }
1815        stmt::Limit::Offset(lo) => {
1816            let limit = as_i64_literal(&lo.limit);
1817            let offset = lo.offset.as_ref().map(as_i64_literal);
1818            Some(Pagination::Offset { limit, offset })
1819        }
1820    }
1821}

1823/// Extracts an `i64` from a `Value::I64` literal expression. Panics on any
1824/// other shape — an invariant violation that `verify` should have caught.
1825fn as_i64_literal(expr: &stmt::Expr) -> i64 {
1826    match expr {
1827        stmt::Expr::Value(stmt::Value::I64(n)) => *n,
1828        _ => panic!("limit/offset must be an i64 literal; got {expr:#?}"),
1829    }
1830}

1832/// Extract the sort direction from a query statement's ORDER BY clause for use
1833/// with `QueryPk` on NoSQL drivers.
1834fn extract_query_pk_order(stmt: &stmt::Statement) -> Option<stmt::Direction> {
1835    let query = stmt.as_query()?;
1836    query.order_by.as_ref().and_then(|ob| {
1837        ob.exprs.first().map(|e| match e.order {
1838            Some(stmt::Direction::Desc) => stmt::Direction::Desc,
1839            _ => stmt::Direction::Asc,
1840        })
1841    })
