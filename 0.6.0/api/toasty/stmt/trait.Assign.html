Trait Assign

    // Required method
    fn into_assignment(self) -> Assignment<T>;

    // Provided method
    fn assign(self, assignments: &mut Assignments, projection: Projection)
       where Self: Sized { ... }
}
Expand description

Apply a field mutation to an update statement’s Assignments map.

Every update builder setter accepts impl Assign<T>. All types that implement IntoExpr<T> also implement Assign<T> with set semantics, so update setters accept the same types as create setters.

For has-many fields, arrays and Vecs implement Assign<List<T>> with set (replace) semantics. Use insert, remove, or apply for incremental mutations.
