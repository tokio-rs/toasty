Trait Scalar 

Source 
pub trait Scalar { }
Expand description

Marks scalar (non-composite) types that are valid as the element type of a model-level Vec<T> collection field. The trait is implemented for every primitive Field (string, integers, uuid, …); Vec<u8> is intentionally not a Scalar so that bytes keep their existing scalar storage and don’t get re-routed through the collection-field path.
