newtype_uuid_get_by_id
Tests get_by_id on a Uuid-inner newtype primary key. The id field lowers to a Record([cast(col, Uuid)]) because the embedded mapping wraps the column in a record and the column’s storage type (e.g. Blob on SQLite) does not match the field’s Uuid type. The per-element decomposition of Record == Record must run during lowering so the cast-handling rule fires; otherwise the cast survives into the SQL serializer.
