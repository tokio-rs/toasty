Struct HasMany 

Source 
pub struct HasMany<T> { /* private fields */ }
Expand description

A lazily-loaded has-many association.

HasMany<T> wraps an optional Vec<T> that is populated when the association is eagerly loaded (via include) or accessed through a generated relation accessor. Before loading, calling get panics.

This type appears as a field on model structs for has-many relations.

Returns a slice of the loaded associated records, or None if the association has not been loaded.

This is the non-panicking counterpart to get. An empty slice means the association is loaded and the record has no related rows; None means the association has not been loaded.
