Expand description

A scope represents a context that contains items of a particular type.

For example, a HasMany<T> is a scope whose items are instances of the target model. The trait provides associated types for building typed paths and create builders within the scope.

Source

fn new_path_root() -> Self::Path<Self::Item>

Return a root path for this scope, anchored at the scope’s model.

This is used by the create! macro to obtain field accessors for nested builders without needing to know the concrete model type.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
