Module operation 

Source
Expand description

Database operations dispatched to drivers.

An Operation is the unit of work sent to Connection::exec. The query engine compiles user queries into one or more Operation values. SQL drivers handle QuerySql and Insert; key-value drivers handle GetByKey, QueryPk, DeleteByKey, FindPkByIndex, UpdateByKey, and (when Capability::scan is true) Scan. Both driver types handle Transaction operations.
